# Pomodoro JS Practice 

Hi! This is a simple pomodoro timer with basic Javascript to implement timer functionality as well as internal counters in local storage that track timers completed, timers skipped, and tomatoes collected. Every timer completion results in a tomato collected at the bottom of the screen by the user.

## Sources Used 
W3Schools, Bro Code (YT), How to WebDev (YT), Stardew Valley (pixel tomato)
